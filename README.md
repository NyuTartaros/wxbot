# WxBot

A WeChat chat bot.

Transfer messages between a specific man 
and Microsoft Xiaoice.  
You need to add Xiaoice before using it.

# Requirements

* python3
* wxpy

# Setup Environment

```bash
python3 -m pip install wxpy
```

# Run

```bash
python3 main.py
```

# Set the specific man info

Edit specific_man.json.
